📌 Project Overview
This project analyzes ride data to provide insights into bookings, revenue, and performance across time, vehicle types, and pickup locations.

The focus is on building a clean in intuitive dashboard that supports fast and effective decision making.

⚙️ Key Technical Features 
DAX measures for Total Revenue, Total Rides, and key KPIs 
Time based analysis by hour and month
Vehicle type performance comparison 
Location based revenue insights
Clean and structured dashboard design

📈 Dashboard Insights
Revenue and demand vary significantly by hour
Certain vehicle types generate higher revenue and usage 
Pickup locatons highlight high demand areas 
Driver ratings remain stable across categories
